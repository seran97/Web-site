# Revaliu Landing

Landing page estatica para Revaliu, lista para publicar en GitHub Pages, Netlify, Vercel estatico o cualquier hosting que sirva archivos HTML/CSS/JS.

## Archivos

- `index.html`: estructura principal del landing.
- `styles.css`: estilos responsivos y visual del sitio.
- `script.js`: contenido renderizado desde arreglos para que sea facil de mantener.

## Publicacion rapida

1. Inicializa o conecta este directorio a un repositorio Git.
2. Sube estos archivos a tu repositorio.
3. Publica la carpeta raiz como sitio estatico.

## Nota

El boton de contacto apunta temporalmente a `contacto@revaliu.com`. Si quieres, en el siguiente paso te lo dejo conectado a tu correo real, WhatsApp o formulario.
